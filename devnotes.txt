The preferred method for development is to use docker. Simply run:
docker-compose --profile develop up  

For production deployment:
docker-compose --env-file .env.production up


To setup the development environment locally (not reccomended):

1. Install Python 3
https://www.python.org/downloads/

2. Install MongoDB
https://docs.mongodb.com/manual/installation/

1. Create a python virtual environment in the project root
# cd /path/to/here/urls.web

Python 2.7 (In case you have python 2 and 3 installed, this will use the correct pip)
# python -m virtualenv ENV

Python 3 (In case you have python 2 and 3 installed, this will use the correct pip)
# python -m venv ENV

Windows Python 3
# py -3 -m venv ENV

2. Activate the Environment
NIX Systems# source ENV/bin/activate
Windows# source ENV/scripts/activate

Note: to exit the virtual environment, just type:
# deactivate

3. Install prereqs
# pip install -r requirements.txt

4. Start the app
# python urls_web/app.py


5. Core Python Modules
PyMongo
requests
dateutils
Flask
Flask-Cors
